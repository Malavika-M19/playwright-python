## Play it Right with Playwright

A beginner-friendly UI test automation project for QA professionals learning Playwright.

This project uses **Playwright with Python** to automate key workflows on the OrangeHRM demo application.

The framework follows the **Page Object Model (POM)** design pattern to keep test code clean, reusable, and maintainable. Since Playwright Python does not include a built-in test runner, **Pytest** is used to organize, execute, and report test cases.

### Technologies Used

* Python
* Playwright
* Pytest
* Page Object Model (POM)
* OrangeHRM Demo Application
